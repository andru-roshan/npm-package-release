# ARJS Component

## Button

### Imports

```
import {Button} from 'arjs-component'
```

### Usage

```
<Button variant="primary" className="classname" label="save" type="button"> </Button>
```

## Chip

### Imports

```
import {Chip} from 'arjs-component'
```

### Usage

```
<Chip className="classname">{String or Jsx elements}</Chip>
```
